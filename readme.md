## ğŸ¨ Customization

### Colors
Each sidebar uses CSS variables for easy customization:

```css
:root {
    --primary-color: #4a90e2;
    --text-primary: #2c3e50;
    --text-secondary: #6c757d;
    --bg-color: #f5f6fa;
}
```

### Dimensions
Adjust sidebar widths and transitions:

```css
:root {
    --sidebar-width: 280px;
    --sidebar-collapsed-width: 70px;
    --transition-speed: 0.3s;
}
```
